Markdown语法学习

既然要写博客，首先就要学习一下Markdown语法
Markdown语法说明（简体中文版

1.标题语法

Markdown支持两种标题语法：类Setext和类atx形式

a.类Setext：

任意数量的’=’或’-‘

b.类Atx：

行首插入1到6个’#’，对应1到6阶，注意’#’ 与标题间要有空格

也可以选择性地闭合类Atx标题：在行尾加上’#’，且数量无需和行首相同

4.代码区块

for (int i = 0; i < 4; i++)
{
    double MobiTem = 0.0;
    x = PosX[0][i];
    y = PosY[0][i];
    vst[x][y]++;
    queenmove[x][y][0] = 1;
    for (int j = 0; j < 8; j++)
    {
        x = PosX[0][i];
        y = PosY[0][i];
        for (int s = 1;; s++)
        {
            x += Dx[j];
            y += Dy[j];
            if (Map[x][y] != BLANK)break;
            vst[x][y]++;
            queenmove[x][y][0] = 1;
            double mobi = 0.0;
            for (int k = 0; k < 8; k++)
            {
                if (Map[x + Dx[k]][y + Dy[k]] == BLANK)
                {
                    mobi += 1.0;
                }
            }
            MobiTem = MobiTem + mobi / s;
        }
    }
    MobilityW += MobiTem;
    if (MobiTem < MobiMin)
    {
        MobiMin = MobiTem;
    }
}
