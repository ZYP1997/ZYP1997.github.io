4.1 最大子数组问题

只有当数组中包含负数时，最大子数组问题才有意义。

暴力求解时间复杂度O(n²)。

使用分治策略的求解方法

设数组A[low…mid…high],mid为中间位置。A的任何连续子数组A[i…j]必然是：low<=i<=j<=mid; mid<i<=j<=high; low<=i<=mid<j<=high。

我们可以在线性时间内求出跨越中点的最大子数组。

时间复杂度O(nlgn)。
